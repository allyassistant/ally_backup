Sub-agent 係一個專業化實例，運行喺自己隔離嘅 context 中，核心概念係「委託」。

每個 Sub-agent 獲得：

• 定義角色嘅 system prompt
• 有限嘅工具集
• 完全隔離嘅 context
• 單一、範圍清晰嘅任務

完成後**只返回最終輸出**，唔返回推理過程或中間步驟。

**Sub-agents 的核心價值：壓縮（compression），唔只係速度**
