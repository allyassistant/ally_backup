## 第一階段：立即實施

### 1. 時間意圖自動捕獲
**目標**：識別用戶嘅時間意圖，自動添加到 Apple Reminders

**實現方式**：
- 創建 `scripts/time_intent_capture.js`
- 觸發詞：「明天」、「後日」、「下周」、「X月X日」
- 整合入強制分析模式，Kimi 識別後調用

### 2. 每週存儲復盤
**目標**：每週自動檢查文件存儲狀態

**實現方式**：
- 創建 `scripts/weekly_storage_review.js`
- 檢查項目：文件位置、備份完整性、Token 使用
- 整合入 HEARTBEAT.md，每週一執行

### 3. 發佈狀態自動更新
**目標**：識別「已發佈」等狀態，自動更新文件位置

**實現方式**：
- 增強現有 Qwen3 腳本
- 觸發詞：「已發佈」、「已發送」、「完成了」
- 自動移動文件到相應目錄

## 第二階段：優化提升

### 4. 人際資源庫
**目標**：記錄客戶、供應商、中介等聯絡資料

**實現方式**：
- 創建 `memory/contacts.json`
- 創建 `scripts/contact_manager.js`
- 功能：添加、查詢、更新、分類

### 5. 記憶提煉機制
**目標**：自動從每日日誌提煉重要內容到 MEMORY.md

**實現方式**：
- 創建 `scripts/memory_distiller.js`
- 每週執行一次
- 識別關鍵：決策、新知識、錯誤教訓

### 6. 月度審閱強化
**目標**：升級現有 monthly_review.js

**實現方式**：
- 增強現有腳本
- 加入統計分析功能
- 生成月度報告

## 第三階段：進階優化

### 7. 三層記憶架構優化
**目標**：將 MEMORY.md 優化為三層結構

**三層結構**：
- **情景記憶**：對話歷史、每日日誌
- **語義記憶**：可復用知識、方法論
- **強制規則**：行為約束、必須遵守嘅規則

### 8. 智能存儲路由加強
**目標**：AI 自動決定文件存儲位置

**實現方式**：
