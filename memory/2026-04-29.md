## v7.9.5 → v8.0.0 Fixes (Kimi Code CLI)

### 1. Color Field Corruption (P0 - CRITICAL)
**Problem:** Fancy shape certs showed "fancy shapes like Pear" in Color field
- #5221657767 (10.04ct Pear) → Color = "fancy shapes like Pear"
- #2524520170 (3.47ct Triangular) → Color = "fancy shapes"

**Root Cause:** AI parsing Comments/Shape and Cutting Style field, placing shape description in Color

**Fix:** Added corruption detection + recovery from rawText/comments
- Check for "fancy shapes like" or "like" pattern
- Fallback to UNKNOWN_FANCY if no real color found

### 2. Internal Graining Penalty (v8.0)
**Problem:** High-clarity stones (VVS1/VVS2) with Internal graining note may appear "not crisp"

**Fix:**
- 10ct+ + graining note: -5 penalty + flag "⚠️ Internal Graining Clarity (10ct+)"
- 5ct+ high clarity (IF/FL/VVS1/VVS2/VS1) + graining note: -2 penalty

### 3. Visual Parallelism Risk (v8.0)
**Problem:** Emerald Cut with non-EX symmetry may show visible step facet misalignment

**Fix:**
- 5ct+ Emerald Cut + Symmetry ≠ EX: -5 penalty + flag "⚠️ Visual Parallelism Risk"
