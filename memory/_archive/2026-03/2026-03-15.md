## 🔧 SSH 連線問題 (2026-03-15)

### 問題
Ally 無法 SSH 到 Bliss

### 原因
- 用錯 username: `ally_assistant` vs `bliss`
- 正確既 username 取決於 remote 機既 user account

### 解決
- Ally → Bliss: `bliss@[TAILSCALE_BLISS_IP]`
- Bliss → Ally: `ally@[TAILSCALE_ALLY_IP]` (or `ally_assistant` depending on Ally's user)

### Lessons
- 每次 set SSH 之前要確認雙方既 username
- HA scripts 既 NODE_ID/PEER_ID 要正確 set
