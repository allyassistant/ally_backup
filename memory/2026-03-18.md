### 2. Ollama 限制記錄 (AGENTS.md 更新)
基於 OpenClaw + Ollama 本地部署踩坑經驗，記錄以下限制：

| 問題 | 影響 | 解決 |
|------|------|------|
| Tool Calling 支援差 | Function calling 成功率低 | 複雜任務用 MiniMax |
| 檔案處理不可靠 | 可能亂改副檔名冒充 | 用 Node scripts/exceljs |
| URL 易造假 | Hallucination 生成假連結 | 所有 URL 要驗證 |
| 繁中搜尋問題 | web_search 可能 422 | 用 web_fetch |

**核心原則**: 本地模型慳 API 費，但時間成本更高
