## CRON `command` Kind Discovery (22:50 HKT)

**🚨 MAJOR FINDING — changes whole cron architecture:**

OpenClaw `CronPayloadSchema` has **3 payload kinds**:
1. `systemEvent` — text injection to main session (no LLM but main session overhead)
2. `agentTurn` — LLM call + isolated session (what all 28 crons currently use)
3. **`command`** — **direct exec, zero LLM call, zero session overhead** ← perfect thin executor

**Command kind fields**: `argv`, `cwd`, `env`, `input`, `timeoutSeconds`, `noOutputTimeoutSeconds`, `outputMaxBytes`

**Verification**: Test cron (`echo`) ran in **19ms, exit 0, zero LLM call** — confirms `command` kind works.
