## 鑽石 4C 標準

### 1. Carat（卡數/重量）

**單位換算**：
- 1 Carat (ct) = 0.2 克 = 200 毫克
- 1 Carat = 100 分 (points)
- 0.5ct = 50分

**重量分級**：
| 分類 | 重量範圍 |
|------|---------|
| 小鑽 | < 0.30ct |
| 中鑽 | 0.30ct - 0.99ct |
| 大鑽 | 1.00ct - 2.99ct |
| 特大鑽 | 3.00ct - 9.99ct |
| 收藏級 | ≥ 10.00ct |

**價格跳躍點**：
- 1.00ct（價格明顯上升）
- 1.50ct
- 2.00ct
- 3.00ct
- 5.00ct
- 10.00ct

### 2. Color（顏色）

**白鑽分級（D-Z）**：
| 等級 | 描述 | 特點 |
|------|------|------|
| D-E-F | 無色 (Colorless) | 最稀有，最昂貴 |
| G-H-I-J | 接近無色 (Near Colorless) | 性價比高 |
| K-L-M | 微黃 (Faint Yellow) | 肉眼可見淡黃 |
| N-R | 很輕微黃 (Very Light Yellow) | 明顯偏黃 |
| S-Z | 輕微黃 (Light Yellow) | 黃色明顯 |

**彩鑽分級**：
- **Faint**（微）
- **Very Light**（很輕微）
- **Light**（輕微）
- **Fancy Light**（淡彩）
- **Fancy**（彩）
- **Fancy Intense**（濃彩）
- **Fancy Vivid**（豔彩）
- **Fancy Deep**（深彩）
- **Fancy Dark**（暗彩）

### 3. Clarity（淨度）

**GIA 淨度分級（由高至低）**：

| 等級 | 名稱 | 含義 |
|------|------|------|
| **FL** | Flawless | 內外無瑕，放大 10 倍都見唔到瑕疵 |
| **IF** | Internally Flawless | 內部無瑕，可能有極微細表面紋理 |
| **VVS1** | Very Very Slightly Included 1 | 極微內含物，專業人員都難發現 |
| **VVS2** | Very Very Slightly Included 2 | 極微內含物，比 VVS1 易發現少少 |
| **VS1** | Very Slightly Included 1 | 微細內含物，專業人員放大 10 倍可見 |
| **VS2** | Very Slightly Included 2 | 微細內含物，比 VS1 明顯 |
| **SI1** | Slightly Included 1 | 小內含物，肉眼可能見到 |
| **SI2** | Slightly Included 2 | 小內含物，肉眼容易見到 |
| **SI3** | Slightly Included 3 | 明顯內含物（部分證書機構使用）|
| **I1** | Included 1 | 內含物明顯，影響光澤 |
| **I2** | Included 2 | 內含物很明顯，嚴重影響光澤 |
| **I3** | Included 3 | 內含物極明顯，可能影響耐用度 |

**實用建議**：
- **投資收藏**：FL-VVS2
- **日常佩戴**：VS1-SI1（性價比最高）
- **預算有限**：SI1-SI2（需挑選肉眼乾淨嘅）

### 4. Cut（切割/車工）

**GIA 切割分級（只適用於圓鑽 RBC）**：

| 等級 | 描述 |
|------|------|
| **EX** | Excellent（極佳）- 反射光線最佳 |
| **VG** | Very Good（很好）- 反射光線很好 |
| **G** | Good（好）- 反射光線良好 |
| **F** | Fair（一般）- 反射光線一般 |
| **P** | Poor（差）- 反射光線差 |

**三項車工指標**：

| 指標 | 英文名 | 說明 |
|------|--------|------|
| **Cut** | 切割比例 | 決定光線反射效果 |
| **Polish** | 拋光 | 表面光潔度 |
| **Symmetry** | 對稱度 | 切面排列對稱性 |

**理想組合**：
- 首選：EX/EX/EX（3EX）
- 次選：EX/EX/VG 或 EX/VG/EX
- 避免：G 以下

**其他形狀嘅車工**：
- 公主方、梨形等異形鑽**無** GIA 切割分級
- 只有 Polish（拋光）同 Symmetry（對稱度）評級

## 證書同分級

### 主要證書機構

| 機構 | 全名 | 特點 |
|------|------|------|
| **GIA** | Gemological Institute of America | 全球最權威，標準最嚴格 |
| **HRD** | Hoge Raad voor Diamant | 比利時，歐洲權威 |
| **IGI** | International Gemological Institute | 國際認可，較常見於商業鑽石 |
| **AGS** | American Gem Society | 美國，以車工評級嚴格聞名 |

### GIA 證書重點

**必須核對**：
1. **Report Number**（證書編號）- 官網可查
2. **Shape and Cutting Style**（形狀同切割）
3. **Measurements**（尺寸）- 長×寬×高 mm
4. **Carat Weight**（重量）
5. **Color Grade**（顏色等級）
6. **Clarity Grade**（淨度等級）
7. **Cut Grade**（車工等級 - 只限圓鑽）
8. **Polish**（拋光）
9. **Symmetry**（對稱度）
10. **Fluorescence**（螢光反應）

### 螢光反應（Fluorescence）

| 等級 | 描述 |
|------|------|
| **None** | 無 |
| **Faint** | 微弱 |
| **Medium** | 中等 |
| **Strong** | 強 |
| **Very Strong** | 極強 |

**影響**：
- 中等以下通常無影響
- Strong/Very Strong 可能令鑽石睇落較蒙（尤其高色級 D-F）
- 部分情況下螢光反而會令鑽石睇落更白

## Rapaport 價格計算

### Rapaport 價格表結構

**形狀分類**：
- **Round**（圓鑽 RBC）
- **Pear**（梨形 - 代表所有異形鑽）

**重量分級**：
| 範圍 | 適用 |
|------|------|
| 0.90-0.99 | 小於 1 卡 |
| 1.00-1.49 | 1 卡至 1.49 卡 |
| 1.50-1.99 | 1.5 卡至 1.99 卡 |
| 2.00-2.99 | 2 卡至 2.99 卡 |
| 3.00-3.99 | 3 卡至 3.99 卡 |
| 4.00-4.99 | 4 卡至 4.99 卡 |
| 5.00-5.99 | **5 卡以上都係呢個表** |

**⚠️ 重要規則**：
> **凡係大過 5 卡嘅鑽石都用 5.00-5.99 個表去計，包括 10 卡、20 卡！**

### 計算公式

```
價格 = (表格值 × 100 × Carat) × (1 - Discount%)
```

**步驟**：
1. 查 Rapaport 表格值（以百元計）
2. × 100 轉為美元
3. × Carat（重量）
4. × (1 - Discount%) 應用折扣

### 計算示例

**Example 1：Pear 5.01 G VS1**
- 表格值（5.00-5.99）：425
- 計算：425 × 100 × 5.01 = $212,925
- 折扣 -15%：$212,925 × 0.85 = **$180,986**

**Example 2：Pear 8.08 K SI1**
- 表格值（5.00-5.99）：145
- 計算：145 × 100 × 8.08 = $117,160
- 折扣 -15%：$117,160 × 0.85 = **$99,586**

**Example 3：RBC 10.02 I SI2**
- 查 Round 10.00-10.99 表
- 表格值：410
- 計算：410 × 100 × 10.02 = $410,820
- 折扣 -15%：$410,820 × 0.85 = **$349,197**

### FL 處理

**FL（Flawless）使用 IF 價格**：
```
FL 價格 = IF 表格值 × 1.1（約 10% 溢價）
```

## 常見問題

### Q: 咩形狀最保值？
A: **圓鑽 RBC** 永遠最保值，流通性最高。其次係 Princess、Emerald。

### Q: 投資應該揀咩等級？
A: 建議 **D-F / IF-VVS2 / 3EX**，重量越大越好。

### Q: SI1 會唔會肉眼睇到瑕疪？
A: 視乎瑕疪位置同類型。建議要求供應商提供淨度圖確認。

### Q: 彩鑽點計 Rapaport？
A: 彩鑽**無** Rapaport 標準定價，由供需決定。Fancy Intense/Vivid 最值錢。

### Q: 10ct 同 5ct 用同一個表，係咪代表 10ct 會平咗？
A: 理論上係，但實際 10ct+ 稀有度高，市場價格通常會高過純粹按 5ct 表計。

### Q: 點分 PR 同 PS？
A: **PR = Princess（公主方）**，**PS = Pear（梨形）**。記住：PR 係方形，PS 係水滴形。
