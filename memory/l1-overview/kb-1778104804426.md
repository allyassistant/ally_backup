### 內容

- **Content Agent** — 創意、起草、編輯、跨平台內容重製
- **Operations Agent** — email 分流、會議準備、每周報告、行政工作

**3️⃣ Research Agent點整**
建立知識庫（競爭對手、行業、客戶profile）→ 給佢 tools（MCP server 接駁 web search、Google Drive、email）→ 設定每週一早上自動運行嘅 workflow。關鍵係三層 prompt：System prompt（角色定義）→ Workflow prompt（每週做乜）→ Output prompt（輸出格式）。

**4️⃣ Content Agent 係最值錢嘅投資**
最費時唔係創意，而係 production work（格式化、寫變體、schedule、追蹤表現）。建立 voice & brand documents（你表現最好嘅 20 篇帖子、style guide）→ 設定 quality gates（voice match、hook strength、value density、originality 評分）→ 低於標準自動重寫。最後你做 20% 靈魂部分，agent 做 80% 生產部分。

**5️⃣ 三個 Agents 協作先係威力所在**
Research agent 發現競爭對手出新功能 → Content agent 立即做三篇回應內容 → Operations agent 準備好接觸受影響客戶嘅 email draft。關鍵係建立 **shared knowledge base**，三個 agent 都可讀寫，呢個先係真正嘅團隊。
