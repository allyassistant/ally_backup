**係咩？**
`memory-wiki` 係 OpenClaw 既 **Knowledge Vault 插件**，幫你將 raw notes 轉變成結構化知識庫。

**核心功能：**

| 功能 | 說明 |
|------|------|
| **Structured Claims** | 將知識拆成「聲稱 + 證據」格式 |
| **Evidence Tracking** | 每個 claim 有 source/provenance |
| **Contradiction Detection** | 自動發現矛盾資訊 |
| **Freshness Tracking** | 顯示邊個資訊已過時 |
| **Dashboards** | 生成視覺化儀表板 |
| **Compiled Digests** | AI 可消費既摘要 |

---

**現有工具：**
- `wiki_search` - 搜尋
- `wiki_get` - 讀取
- `wiki_apply` - 應用
- `wiki_lint` - 質量檢查
