# Excel 鑽石估值報表技巧

## ============================================
## 技巧一：建立自動化估值計算表
## ============================================

### 工作表結構設計

【Sheet 1: 鑽石資料輸入】
```
A欄: 項目        B欄: 數值          C欄: 公式/說明
--------------------------------------------------
報告編號          VAL-2026-0210-001
日期              2026-02-10
客戶名稱          [輸入]

鑽石 1 資料:
形狀              RBC              ← 下拉選單
克拉              1.05
顏色              G                ← 下拉選單
淨度              VS2              ← 下拉選單
切工              EX               ← 下拉選單
拋光              EX               ← 下拉選單
對稱              EX               ← 下拉選單
螢光              None             ← 下拉選單
證書              GIA              ← 下拉選單

Rapaport 基準價   7800             ← 手動輸入或 VLOOKUP
市場折扣          -28%             ← 百分比格式

計算結果:
調整後每卡價      =B15*(1+B16)     ← 公式示例
總估值(USD)       =B18*B4
總估值(HKD)       =B19*7.8
```

### 關鍵公式

【1. 下拉選單製作】
- 選擇儲存格 → 資料 → 資料驗證 → 清單
- 形狀清單：RBC,PR,EM,OV,CU,RAD,HS,MQ
- 顏色清單：D,E,F,G,H,I,J,K,L,M
- 淨度清單：FL,IF,VVS1,VVS2,VS1,VS2,SI1,SI2,I1,I2,I3
- 等級清單：EX,VG,G,F,P

【2. 自動查找 Rapaport 價格 (VLOOKUP)】
```excel
=VLOOKUP(B5&B6, Rapaport_Table!A:D, 4, FALSE)
```
- B5 = 顏色, B6 = 淨度
- 需要預先建立查找表

【3. 折扣計算公式】
```excel
=基準價 * (1+切工調整) * (1+螢光調整) * (1+證書調整) * (1+市場折扣)
```

【4. 條件判斷 (IF / IFS)】
```excel
=IF(B7="EX", IF(B8="EX", IF(B9="EX", 0.08, 0), -0.05), -0.10)
```
- 判斷是否 3EX，給予不同調整

## ============================================
## 技巧二：批量庫存估值表
## ============================================

【Sheet: 庫存清單】
```
A      B      C      D      E      F      G      H      I      J      K
編號   形狀   克拉   顏色   淨度   切工   拋光   對稱   螢光   證書   基準價
001    RBC    1.02   G      VS1    EX     EX     EX     None   GIA    8200
002    PR     0.85   F      VVS2   -      VG     VG     Faint  GIA    4800
003    EM     1.50   H      VS2    -      EX     EX     None   GIA    6500
```

【自動計算欄位】
```excel
L2 (切工調整): =IF(B2="RBC", IF(AND(F2="EX",G2="EX",H2="EX"),0.08,IF(AND(F2="VG",G2="VG",H2="VG"),-0.05,0)),0)

M2 (螢光調整): =IF(I2="None",0,IF(I2="Faint",-0.01,IF(I2="Medium",-0.03,IF(I2="Strong",-0.10,-0.18))))

N2 (證書調整): =IF(J2="GIA",0,IF(J2="IGI",-0.05,IF(J2="HRD",-0.03,-0.20)))

O2 (市場折扣): -0.28  (可設為全域變數)

P2 (最終每卡價): =K2*(1+L2+M2+N2)*(1+O2)

Q2 (總估值USD): =P2*C2

R2 (總估值HKD): =Q2*7.8
```

## 技巧四：客戶報價單模板
## ============================================

【報價單格式】
```
┌─────────────────────────────────────────┐
│           鑽石報價單 QUOTATION          │
│           [公司名稱]                    │
├─────────────────────────────────────────┤
│ 報價編號：QT-20260210-001               │
│ 日期：2026年2月10日                     │
│ 客戶：________________                  │
│ 有效期：7日                             │
├─────────────────────────────────────────┤
│ 項次 │ 規格              │ 數量 │ 單價    │ 金額    │
│ 1    │ 1.05ct G VS2 3EX  │ 1    │ $6,365  │ $6,365  │
│ 2    │ 0.85ct F VVS2     │ 1    │ $2,684  │ $2,684  │
├─────────────────────────────────────────┤
│ 小計：                                    USD 9,049   │
│ 折扣：                                    USD   450   │
│ 合計：                                    USD 8,599   │
│ 港幣約：                                  HKD 67,072  │
├─────────────────────────────────────────┤
│ 備註：                                  │
│ • 價格基於 Rapaport 2026年2月基準       │
│ • 包含 GIA 證書                         │
│ • 不含鑲嵌費用                          │
└─────────────────────────────────────────┘
```

【Excel 公式】
```excel
小計: =SUM(E2:E10)
折扣: =IF(小計>10000, 小計*0.05, 小計*0.03)  ← 滿萬送5%，否則3%
合計: =小計-折扣
港幣: =合計*7.8
```
