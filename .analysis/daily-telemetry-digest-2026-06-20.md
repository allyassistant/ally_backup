## What was built

### New: `scripts/daily_telemetry_digest.js` (350 lines)

5 sections + footer, formatted as Discord markdown (846 chars, well under 2000 limit):

1. **🛡️ Audit & Repair**
   - Issues found (total + by tier: production/utility/debug)
   - Auto-fixes OK
   - New proposals (added + pending)
   - Cross-script follow-ups (Layer 2)

2. **🎯 Skill Pipeline Health**
   - Junk-in-Production (target <10%) ✅/❌
   - Validator Catch (target ≥25%) ✅/❌
   - passedAndQuarantined (LLM-overridden)
   - LLM override active count

3. **📨 Skill Review Queue**
   - Pending entries (v=2 vs v=3)
   - Oldest entry age

4. **🔄 Self-Healing Loop**
   - Fixes applied (count + unique files)
   - Verify fails
   - Avg fix time

5. **📚 Skill Usage**
   - used / skipped / rejected counts
   - Top 3 used skills
   - Top 3 skipped skills
