# 內容分類指南

當你要加新內容入 AGENTS.md，用呢個 quick check 決定擺邊度：

| 新內容係咩？ | 放邊個 section | 例子 |
|--------------|---------------|------|
| 收到 msg 後立即要決定嘅行為 | **對話行為** | 有新 tool route、新 reply rule |
| 做 task 時嘅思考框架、code 規則 | **思考框架** | 新 spawn pattern、新 code rule |
| Reference、設定、必做 checklist | **背景設定** | 新 SOP、新 checklist step |
| **唔肯定？** | 放背景設定，之後 review | — |

## 原則
- 如果係「見到 msg 就要做嘅」→ 對話行為
- 如果係「做 task 時要諗嘅」→ 思考框架
- 如果係「setup / reference / 背景」→ 背景設定
