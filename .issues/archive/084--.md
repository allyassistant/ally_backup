## 目標

建立「驗證閉環實驗」系統：

```
1. auto_fix.js 修復後，記錄修復內容 + 預期效果
2. 24 小時後檢查 errors.json 有無同類錯誤再出現
3. 計算「修復成功率」
4. 成功率 < 50% → 自動降級
```
