Mac Excel 2019 條件格式設定教學
================================

由於 ExcelJS 生成嘅條件格式同 Mac Excel 2019 有相容性問題，
請依照以下步驟手動設定「打 Y 變綠色」功能：

方法：手動添加條件格式（推薦）
================================

步驟 1：選擇範圍
----------------
1. 開啟 stock_mac.xlsx
2. 點一下儲存格 A2（第一個 Checked 欄）
3. 按 Shift + Command + ↓（向下箭頭）選擇整個 A 欄數據
4. 或者手動拖曳選擇 A2 到 A136（全部 135 粒石）

步驟 2：開啟條件格式設定
------------------------
1. 點擊頂部選單「格式」
2. 選擇「條件式格式設定...」
3. 在彈出視窗中點擊「+」新增規則

步驟 3：設定規則
----------------
1. 「樣式」選擇「經典」
2. 條件選擇「只格式化包含下列的儲存格」
3. 左邊下拉選單選「等於」
4. 右邊輸入框打：Y
5. 點擊「格式...」按鈕

步驟 4：選擇顏色
----------------
1. 在「填滿」分頁
2. 選擇綠色（建議選淺綠色，如「綠色，淺色 80%」）
3. 撳「確定」

步驟 5：完成設定
----------------
1. 撳「確定」關閉格式視窗
2. 撳「確定」關閉條件格式視窗

測試
----
在任意 Checked 欄打 Y，應該會看到儲存格變綠色！

=======================================

進階：整行變綠色（可選）
========================

如果你想打 Y 時整行都變綠色：

步驟 1：選擇範圍
----------------
1. 選擇 A2 到 P136（全部數據，包括所有欄）

步驟 2：新增公式規則
--------------------
1. 格式 → 條件式格式設定
2. 點擊「+」新增規則
3. 「樣式」選擇「經典」
4. 改為「使用公式決定要格式化哪些儲存格」
5. 輸入公式：=$A2="Y"
   （注意：一定要有 $ 符號在 A 前面！）

步驟 3：設定格式
----------------
1. 點擊「格式...」
2. 填滿分頁選綠色
3. 確定

步驟 4：完成
------------
撳確定，完成設定！

=======================================

提示
====

• 如果 Mac Excel 2019 冇「條件式格式設定」選項，
  可能係你嘅 Excel 版本較舊或設定唔同，
  建議更新到最新版本或使用簡單文字標記（Y/N）

• 設定好一次後，儲存 Excel 檔案，下次開啟會保留設定

• 可以為不同狀態設定唔同顏色：
  - Y = 綠色（已點/有貨）
  - N = 紅色（無貨）
  - R = 黃色（已留/Reserved）
