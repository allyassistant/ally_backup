## 🔷 第一部分：4C 標準

### Cut (切工)

**定義：** 將鑽石原石轉變為精美拋光寶石的工藝過程

**圓形明亮式切割包含：**
- 57-58 個刻面
- 桌面 (Table)、星瓣、下腰面、冠部、主刻面、尖底

**切工比例黃金範圍 (Excellent)：**
| 參數 | 理想範圍 |
|------|----------|
| 桌面百分比 | 53%-58% |
| 總深度百分比 | 59%-62.5% |
| 冠角 | 34°-35° |
| 亭角 | 40.6°-41.8° |

---

### Color (顏色)

**GIA 顏色等級系統 (D-Z)：**

| 等級 | 分類 | 描述 |
|------|------|------|
| D-F | 無色 (Colorless) | 幾乎沒有任何顏色 |
| G-J | 近無色 (Near Colorless) | 肉眼下難以察覺 |
| K-M | 微黃 (Faint Yellow) | 肉眼可見淡淡黃/棕色調 |
| N-R | 很淡黃 (Very Light Yellow) | 明確可見的黃/棕色調 |
| S-Z | 淡黃 (Light Yellow) | 明顯的黃色或棕色調 |

**為什麼從 D 開始？**
- 避免與舊系統（A、B、C）混淆
- 建立全新的統一標準

---

### Clarity (淨度)

**GIA 淨度等級（11個）：**

| 等級 | 簡稱 | 定義 |
|------|------|------|
| Flawless | FL | 10倍放大鏡下無內含物或表面特徵 |
| Internally Flawless | IF | 10倍放大鏡下無內含物 |
| Very Very Slightly Included | VVS1/VVS2 | 極難見到極微小內含物 |
| Very Slightly Included | VS1/VS2 | 難以見到微小內含物 |
| Slightly Included | SI1/SI2 | 容易見到內含物，肉眼通常不可見 |
| Included | I1/I2/I3 | 內含物肉眼可見 |

**常見內含物類型：**
| 術語 | 描述 |
|------|------|
| 羽裂紋 (Feather) | 類似羽毛狀的裂紋 |
| 雲狀物 (Cloud) | 多個極細小內含物聚集 |
| 針狀物 (Needle) | 細長的晶體 |
| 晶體 (Crystal) | 被困在鑽石內的小晶體 |
| 洞痕 (Cavity) | 表面的小凹陷 |

---

### Carat (克拉)

**定義：** 重量單位，1克拉 = 0.2克 = 200毫克

**克拉與分：**
- 0.50ct = 50分 = 半克拉
- 0.75ct = 75分
- 0.25ct = 25分

**克拉與尺寸對照：**
| 克拉重量 | 大約直徑 |
|----------|----------|
| 0.25 ct | 4.1 mm |
| 0.50 ct | 5.0 mm |
| 1.00 ct | 6.4 mm |
| 2.00 ct | 8.1 mm |

**魔術尺寸：** 0.50ct、0.70ct、0.90ct、1.00ct、1.50ct、2.00ct

## 🔷 第二部分：鑽石類型系統

### 兩大類型

| 類型 | 描述 | 佔比 |
|------|------|------|
| **Type I** | 含有可檢測到的氮原子 | ~98% |
| **Type II** | 不含可檢測到的氮或硼 | ~1-2% |

### Type I 細分類

| 細分類 | 氮形式 | 顏色影響 |
|--------|--------|----------|
| Type IaA | A聚集體（兩氮原子配對） | 無明顯顏色 |
| Type IaB | B聚集體（多氮原子圍繞空位） | 輕微黃/棕色 |
| Type IaAB | 混合型 | 最常見 |
| Type Ib | 分散單原子氮 | 明顯黃/棕橙色 |

### Type II 細分類

| 細分類 | 描述 | 特徵 |
|--------|------|------|
| **Type IIa** | 最純淨，不含氮或硼 | 無色或彩色，無熒光 |
| **Type IIb** | 含硼 | 藍色，半導體特性 |

## 🔷 第三部分：光學表現

### Brilliance (光彩)
白光從冠部返回的總量，取決於切工比例、拋光品質、淨度

### Fire (火彩)
白光被分解成彩虹光譜（紅橙黃綠藍紫）的現象，源於色散 (Dispersion)

### Scintillation (閃爍)
觀察者或光源移動時，刻面明暗交替變化的效果

## 🔷 第四部分：處理方式

| 處理方式 | 說明 | 穩定性 |
|----------|------|--------|
| 雷射鑽孔 (Laser Drilling) | 燒蝕通道直達深色內含物 | 永久，不可逆 |
| 裂縫填充 (Fracture Filling) | 填充透明材料減少裂縫可見度 | 較差，怕熱和化學品 |
| HPHT 處理 | 高溫高壓改善顏色 | 穩定 |

## 🔷 第五部分：合成鑽石

### 生產方法

| 方法 | 全稱 | 特點 |
|------|------|------|
| **HPHT** | High Pressure High Temperature | 模擬天然形成環境，可能含金屬內含物 |
| **CVD** | Chemical Vapor Deposition | 低壓沉積，通常為 Type IIa |

### 識別方法

| 特徵 | HPHT | CVD |
|------|------|-----|
| 典型類型 | Ib 或 IIa | IIa |
| 內含物 | 金屬催化劑 | 可變，有時有空洞 |
| 磁性 | 可能有 | 通常無 |
| 典型熒光 | 十字圖案 | 弱橙紅或無 |

---

## 🔷 第六部分：熒光反應

### GIA 熒光等級

| 等級 | 描述 |
|------|------|
| None | 完全無熒光 |
| Faint | 強烈UV光下極微弱發光 |
| Medium | 標準UV光下明確發光 |
| Strong | 標準UV光下非常明亮 |
| Very Strong | 正常日光下都可能發光 |

### 熒光對不同顏色的影響

| 顏色等級 | 藍色熒光建議 | 理由 |
|----------|-------------|------|
| D-E (無色) | None-Faint | 避免朦朧 |
| F-G (近無色) | None-Medium | 中性至輕微正面 |
| H-I (微黃) | Medium-Strong | 抵消黃色 |
| J-N (淡黃) | Medium-Strong | 減少黃色視覺 |

## 📝 模擬試題精選

### 選擇題

**Q1: GIA顏色分級從D開始的主要原因？**
- A) D代表"Diamond"
- **B) 與現有系統區分，避免混淆** ✅
- C) D色是最高等級
- D) A、B、C被用壞了

**Q2: 1克拉 = ?**
- A) 100毫克
- **B) 200毫克** ✅
- C) 0.1克
- D) 0.5克

**Q3: VS1淨度在10倍放大鏡下？**
- A) 完全看不到內含物
- B) 容易看到內含物
- **C) 難以看到內含物** ✅
- D) 肉眼可見內含物

### 填充題

1. 4C = Cut、Color、Clarity、**Carat**
2. 圓形明亮式切割通常有 **57或58** 個刻面
3. GIA淨度最高兩個等級是 FL 和 **IF (Internally Flawless)**
4. HPHT = **高溫高壓處理** (High Pressure High Temperature)
5. 大多數天然鑽石 (~98%) 屬於 Type **I**

### 是非題

1. **錯** - 兩顆相同4C的鑽石價格不一定相同（熒光、內含物位置、品牌等都會影響）
2. **對** - Type Ib合成鑽石比Type IIa更常見（HPHT早期主要是Type Ib）
3. **對** - GIA報告會標註所有熒光強度，包括None
