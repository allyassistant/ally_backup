## 🔷 第一部分：GIA考試格式

### 考試結構

| 模組 | 內容 | 考試形式 |
|------|------|----------|
| 鑽石 (Diamonds) | 4C分級、合成鑽石、處理方式 | 筆試 + 實踐 |
| 有色寶石 (Colored Stones) | 彩色寶石、產地、處理 | 筆試 + 實踐 |
| 儀器使用 (Instrumentation) | 各種鑑定儀器的操作 | 實踐考試 |
| 零售珠寶 (Retail Jewellery) | 商業實踐、市場知識 | 筆試 |

### 分數要求

- 每個模組需要達到 **75%** 或以上才能通過
- 如有不及格，可以申請補考
- 完成所有模組後才能獲得GG文憑

### 答題技巧

| 題型 | 技巧 |
|------|------|
| 選擇題 | 仔細閱讀，注意否定詞；排除明顯錯誤選項 |
| 填充題 | 確保術語拼寫正確；使用標準術語 |
| 是非題 | 注意「總是」、「從不」等絕對性詞語 |
| 簡答題 | 回答簡潔明確；使用要點形式 |

### 寶石識別流程

```
肉眼觀察 → 折射率測量 → 偏光儀測試 → 比重測量 → 顯微觀察 → 得出結論
```

### 常見錯誤提醒

| 錯誤類型 | 示例 |
|----------|------|
| 混淆相似術語 | 色散 vs 雙折射；光彩 vs 火彩 vs 閃爍 |
| 記錯數值 | 1克拉 = 0.2克 = 200毫克 |
| 忽略單位 | 折射率是無單位的數值 |
| 理解偏差 | 「無處理」不等於「最好」 |

### 問答題精選

**問題1：解釋4C中每個C的重要性，哪個最重要？**

| C | 重要性 | 說明 |
|---|--------|------|
| 切工 (Cut) | **★★★★★** | 直接影響光彩、火彩和閃爍，即使其他4C完美，切工不佳也會使鑽石暗淡 |
| 顏色 (Color) | ★★★★☆ | 影響視覺吸引力，D-F無色，等級越高越昂貴 |
| 凈度 (Clarity) | ★★★☆☆ | 大多數內含物需要放大才能看到，SI以上在肉眼下差異不大 |
| 克拉 (Carat) | ★★★☆☆ | 主要影響尺寸，非線性溢價，存在「魔術尺寸」 |

**結論：** 最重要是切工，因為它直接決定鑽石的光學表現。

## 🔷 第五部分：考前準備

### 考前一周

- 系統回顧所有筆記
- 重點複習容易忘記的數值和定義
- 做模擬試題熟悉題型
- 確保了解考試格式和時間

### 考前一天

- 準備好准考證和身份證件
- 準備好學習工具（放大鏡、筆等）
- 早點休息，保證充足睡眠
- 避免臨時抱佛腳

### 考試當天

1. 提前到達考場
2. 帶齊所有必需物品
3. 深呼吸，保持冷靜
4. 先快速瀏覽試卷
5. 先答有把握的題目
6. 留時間檢查答案

### 心態調整

- 相信自己的準備
- 不要過度緊張
- 遇到難題不要慌
- 保持積極的態度
- 記住：這只是學習的一部分
