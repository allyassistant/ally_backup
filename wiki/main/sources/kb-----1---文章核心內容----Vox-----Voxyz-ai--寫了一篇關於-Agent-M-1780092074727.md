### 1️⃣ 文章核心內容

**Vox** (@Voxyz_ai) 寫了一篇關於 Agent Memory 的深度文章。核心論點：加大 Context 解決唔到「Agent 失憶」問題 — 你需要係 **Wiki (跨對話)** + **錄音 (同對話內)**，而唔係更大的桌面。

---

### 2️⃣ 五大要點

**1. Context 加大只係桌面變大，唔會畀 Agent 一個 Wiki**

> Bigger context only makes the desk wider. It doesn't give the agent a wiki, and it doesn't give it a recording.

加大 Context Window 只係將所有紙放喺枱面，但 Agent 唔識系統性搵野。真正缺嘅係兩個嘢：跨對話嘅知識庫 + 同對話內嘅完整記錄。

**2. GBrain = 公司 Wiki / Notion（跨對話知識庫）**

- 儲存「跨任何對話都成立的事實」：人、項目、客戶、政策、決定
