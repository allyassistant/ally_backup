### 2️⃣ 五大要點

**1. Agent 做完一輪就停，唔係 prompt 問題，係 runtime 問題**

> 長期工作的秘密不是「不斷說繼續」，而是 Gateway + Cron + Heartbeat + 狀態文件。

Agent 話「繼續工作」然後停，係因為多數 Agent 產品嘅交互單位係 **turn** — 做完一輪唔會有外部調度器再叫醒佢。

**2. 四層機制：Gateway → Cron → Heartbeat → 狀態文件**

- **Gateway**：真正嘅後台鬧鐘，冇 Gateway 嘅話 cron 任務只係「寫好咗但冇人睇錶」
