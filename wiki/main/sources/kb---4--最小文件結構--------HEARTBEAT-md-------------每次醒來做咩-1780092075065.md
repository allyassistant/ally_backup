**4. 最小文件結構**

```
HEARTBEAT.md           → 每次醒來做咩
config/continuity_policy.md → 長期運行規則
memory/current-state.md → 當前進展（階段/主任務/已完成）
memory/task-queue.md    → NOW / NEXT / LATER / BLOCKED / DONE
memory/run-state.md     → 上一輪接力訊息
logs/                   → 運行記錄
```
核心原則：**Agent 可以失去聊天上下文，但不能失去工作上下文。**
