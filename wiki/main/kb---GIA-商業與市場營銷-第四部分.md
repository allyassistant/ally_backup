## 🔷 第一部分：寶石定價 (Pricing)

### 價值構成要素

| 要素 | 說明 |
|------|------|
| 稀有性 (Rarity) | 稀缺程度直接影響價格 |
| 美觀 (Beauty) | 視覺吸引力是最基本的要求 |
| 耐久性 (Durability) | 能夠長期保存的特點 |
| 可接受性 (Acceptability) | 市場需求和文化認可 |

### 價格 vs. 價值的區別

| 術語 | 定義 |
|------|------|
| 價格 (Price) | 實際成交的金額 |
| 價值 (Value) | 客戶認為值得支付的程度 |
| 成本 (Cost) | 生產或獲取商品所付出的代價 |

### 彩色寶石定價模型

**4C 在彩色寶石中的權重：**

| 4C | 對價格的影響程度 |
|---|-----------------|
| Color (顏色) | ★★★★★ (50-80%的價值決定因素) |
| Carat (克拉) | ★★★★☆ (非線性溢價效應) |
| Clarity (淨度) | ★★★☆☆ (視寶石類型而定) |
| Cut (切工) | ★★★☆☆ (影響顏色展現和重量) |

### 定價策略

| 方法 | 公式 | 特點 |
|------|------|------|
| 成本加成法 (Cost-Plus) | 零售價 = 成本 × (1 + 加成百分比) | 簡單直接 |
| 市場導向法 (Market-Oriented) | 零售價 = 市場行情 × 調整係數 | 考慮品牌、服務因素 |
| 價值導向法 (Value-Based) | 零售價 = 客戶認知價值 - 價格敏感度 | 強調創造附加價值 |

### 稀有性溢價

| 稀有情況 | 溢價幅度 |
|----------|----------|
| 頂級顏色 | 100-500%+ |
| 大克拉 | 30-200% per carat |
| 無處理 | 50-100% |
| 特殊產地 | 50-300% |

## 🔷 第二部分：零售管理 (Retail Management)

### 珠寶零售店的管理要素

**陳列設計原則：**
- 焦點突出：重要商品放在視覺焦點位置
- 分類清晰：按寶石類型或價格分區
- 光線充足：珠寶需要良好的照明來展現光彩
- 安全展示：防盜、防損壞措施

**照明的重要性：**
- 使用接近自然光的照明
- 珠寶展示燈通常使用 5000K-6500K 的色溫
- 避免眩光，保護客戶眼睛

### 客戶關係管理

| 客戶類型 | 需求重點 | 銷售策略 |
|----------|----------|----------|
| 投資型 | 保值增值、稀有性 | 強調稀缺性和市場趨勢 |
| 情感型 | 紀念意義、外觀 | 強調設計和情感價值 |
| 實用型 | 性價比、日常佩戴 | 強調耐久性、易搭配 |
| 收藏型 | 珍貴稀有、獨特性 | 強調獨一無二的特點 |

### 行銷與推廣

**數位行銷渠道：**
- Instagram/Pinterest：視覺驅動的平台
- Facebook：建立社群和活動宣傳
- 小紅書：中國市場重要的珠寶行銷平台
- 微信：私域流量和客戶關係管理

**線下行銷：**
- 珠寶展覽和鉴赏会
- 設計師見面會
- VIP客户专属活动
- 节日主题活动

## 🔷 第三部分：珠寶商業 (Jewelry Business)

### 行業鏈

```
礦區/開採 → 原石交易 → 切割加工 → 批發分銷 → 零售 → 消費者
```

### 各環節的角色

| 環節 | 主要功能 | 利潤空間 |
|------|----------|----------|
| 開採商 | 開採原石 | 視乎礦源控制 |
| 原石交易商 | 購買和銷售原石 | 5-20% |
| 切割工廠 | 切割和拋光 | 10-30% |
| 成品批發商 | 分銷成品寶石 | 10-25% |
| 珠寶製造商 | 設計和製作珠寶 | 20-50% |
| 零售商 | 面對消費者銷售 | 50-200%+ |

### 主要交易中心

| 城市 | 特色 |
|------|------|
| 安特衛普 (Antwerp) | 鑽石交易之都 |
| 孟買 (Mumbai) | 彩色寶石切割中心 |
| 曼谷 (Bangkok) | 紅寶石、藍寶石貿易 |
| 可倫坡 (Colombo) | 藍寶石交易中心 |
| 香港/中國 | 翡翠和亞洲珠寶市場 |

### 金屬材料選擇

| 金屬 | 特點 | 市場地位 |
|------|------|----------|
| 鉑金 (Platinum) | 稀有、耐久、不變色 | 高端 |
| 18K金 | 耐久、色澤好 | 高端主流 |
| 14K金 | 硬度高、價格適中 | 中高端 |
| 9K金 | 價格實惠 | 中低端 |
| 925銀 | 價格親民、易氧化 | 大眾市場 |

### 珠寶電子商務策略

| 策略 | 說明 |
|------|------|
| 信任建設 | 清晰的退換政策、第三方檢測證書、詳細的產品描述 |
| 視覺呈現 | 高質量產品攝影、多角度展示、視頻呈現、AR/VR試戴技術 |
| O2O結合 | 線上下單線下取貨、線上預約線下體驗 |

### 品牌定位

| 品牌類型 | 代表 | 特點 |
|----------|------|------|
| 高端奢侈品牌 | Cartier, Tiffany & Co. | 頂級品質、悠久歷史、強勢品牌 |
| 設計師品牌 | 獨立珠寶設計師 | 獨特設計、藝術價值、個人風格 |
| 時尚珠寶品牌 | Pandora, Swarovski | 時尚感強、價格親民、更新快速 |
| 傳統珠寶品牌 | 周大福、周生生 | 傳統工藝、市場認可、渠道優勢 |

## 🔷 第四部分：道德採購 (Ethical Sourcing)

### 什麼是道德採購

道德採購是指珠寶企業在採購過程中，確保其產品來自負責任的供應鏈，不涉及剝削、污染或不當行為：

- 環境保護
- 工人權益
- 社區發展
- 人權保障
- 衝突預防

### 血鑽石與金伯利流程

**血鑽石 (Blood Diamond / Conflict Diamond)：**
- 在戰亂地區開採
- 用於資助武裝衝突
- 侵犯人權

**金伯利進程 (Kimberley Process)：**
- 成立於2003年
- 參與國必須對毛坯鑽石進出口進行監管
- 每批毛坯鑽石必須附有金伯利進程證書

**局限性：**
- 只適用於毛坯鑽石，不包括成品鑽石
- 只定義「衝突鑽石」，不涵蓋其他道德問題

### 主要道德採購標準

| 標準/認證 | 適用範圍 | 主要內容 |
|-----------|----------|----------|
| 金伯利進程 | 鑽石 | 衝突鑽石預防 |
| RJC (責任珠寶業委員會) | 所有寶石和金屬 | 全面道德、社會、環境標準 |
| Fairtrade Gold | 黃金 | 公平貿易認證 |
| Fairmined Gold | 黃金 | 負責任採礦認證 |

### RJC 認證

**責任珠寶業委員會 (Responsible Jewellery Council)：**

涵蓋範圍：
- 人權和勞工權利
- 環境管理
- 珠寶產品披露
- 供應鏈盡職調查

### 實驗室培育寶石的道德優勢

- 不需要採礦，減少環境破壞
- 不涉及衝突地區問題
- 供應鏈透明可追溯
- 價格較天然寶石實惠

## 🔷 第五部分：模擬試題

### 選擇題

**Q1: 彩色寶石定價中最重要的因素是？**
- A) 克拉重量
- **B) 顏色飽和度** ✅ (50-80%的價值)
- C) 淨度
- D) 切工

**Q2: 「血鑽石」是指？**
- **B) 在武裝衝突地區開採、用於資助衝突的鑽石** ✅
- 解析：與顏色或品質無關，與來源和用途相關

**Q3: 專門負責珠寶行業道德採購認證的機構是？**
- A) GIA
- **B) RJC (責任珠寶業委員會)** ✅
- C) ISO
- D) FTC

**Q4: 珠寶店照明應該使用？**
- **C) 接近自然光的 5000K-6500K 色溫** ✅
- 暖色調會改變寶石真實顏色

**Q5: 實驗室培育寶石？**
- **B) 沒有任何道德爭議** ✅
- 沒有採礦帶來的環境破壞和衝突問題

### 填充題

1. 珠寶定價的基本方法包括成本加成法、市場導向法和 **價值 (Value-Based)** 導向法
2. 金伯利進程是針對 **毛坯鑽石 (Rough Diamond)** 的國際貿易管制機制
3. RJC 是專門負責珠寶行業 **道德、社會和環境** 標準的機構
4. 珠寶店陳列中，照明應該使用接近自然光的 **5000K-6500K** 色溫
5. 實驗室培育寶石的兩種主要生產方法是 HPHT 和 **CVD (化學氣相沉積)**

### 是非題

1. **錯** - 彩色寶石的定價與鑽石不同，顏色是最重要的因素（50-80%）
2. **錯** - 道德採購涵蓋更廣泛的範圍，不僅是衝突鑽石
3. **錯** - 實驗室培育寶石必須向消費者明確披露
