## 🔷 第二部分：折射儀 (Refractometer)

### 什麼是折射率 (Refractive Index, RI)

折射率是光線進入寶石時速度變化的量度，是識別寶石最重要的物理常數之一。

**基本原理：** 當光線從空氣（RI ≈ 1.0）進入寶石時會發生折射，折射角度取決於寶石的折射率。折射儀利用臨界角 (Critical Angle) 原理來測量這個數值。

### 測量步驟

1. 清潔寶石和折射儀半球
2. 在半球上滴一滴接觸液（通常為二碘甲烷，RI = 1.74）
3. 將寶石放在半球上，接觸面朝下
4. 從目鏡觀察，讀取明暗交界線的刻度

### 常見寶石的 RI 值

| 寶石 | RI 範圍 | 光學性質 |
|------|---------|----------|
| 鑽石 (Diamond) | 2.417 | 均質體 (Isotropic) |
| 紅寶石 (Ruby) | 1.762-1.770 | 非均質體 (Anisotropic) |
| 藍寶石 (Sapphire) | 1.762-1.770 | 非均質體 (Anisotropic) |
| 祖母綠 (Emerald) | 1.577-1.583 | 非均質體 (Anisotropic) |
| 水晶 (Quartz) | 1.544-1.553 | 非均質體 (Anisotropic) |
| 立方氧化鋯 (CZ) | 2.15-2.18 | 均質體 (Isotropic) |

### 單折射 (SR) vs 雙折射 (DR)

| 類型 | 說明 | 例子 |
|------|------|------|
| **單折射 (SR)** | 均質體，光線在所有方向傳播速度相同，只有一個 RI 值 | 鑽石、立方氧化鋯、石榴石、玻璃 |
| **雙折射 (DR)** | 非均質體，光線進入後分裂為兩條，有兩個或三個 RI 值 | 所有非等軸晶系寶石（紅寶石、藍寶石、祖母綠等） |

**雙折射率計算：** DR = RI最大值 - RI最小值

## 🔷 第三部分：光譜儀 (Spectroscope)

### 什麼是光譜儀

光譜儀是將白光分解為其組成顏色（光譜）的儀器，用於觀察寶石吸收光譜 (Absorption Spectrum)。

**基本原理：** 當白光通過寶石時，某些波長的光被吸收，剩餘的光形成吸收光譜。吸收線和吸收帶的位置揭示了寶石的化學成分和內部結構。

### 常見寶石的光譜特徵

| 寶石 | 致色元素 | 光譜特徵 |
|------|---------|----------|
| **紅寶石 (Ruby)** | 鉻 (Chromium) | 694.2nm 強吸收線（最特徵）、692.8nm、668nm、659nm |
| **祖母綠 (Emerald)** | 鉻 (Chromium) | 683nm 強吸收線、637nm、606nm 寬吸收帶 |
| **藍寶石 (Sapphire)** | 鐵 (Iron) | 450nm 寬吸收帶、460nm、470nm 吸收線 |
| **黃鑽** | 氮 (Nitrogen) | 415nm 吸收線 |
| **藍鑽** | 硼 (Boron) | 500nm 吸收帶 |

## 🔷 第四部分：比重測定 (Specific Gravity)

### 什麼是比重

比重是寶石密度與水密度（4°C 時為 1 g/cm³）的比值。

**計算公式：**
```
比重 (SG) = 寶石重量 / 同體積水的重量
= 寶石在空氣中的重量 / (寶石在空氣中的重量 - 寶石在水中的重量)
```

### 測定方法

| 方法 | 說明 | 優缺點 |
|------|------|--------|
| **水重法 (Hydrostatic)** | 使用精確天平測量空氣和水中重量 | 最準確，需要計算 |
| **重液法 (Heavy Liquids)** | 觀察寶石在已知比重液體中的浮沉 | 快速直觀，有毒性 |

### 常見寶石的比重

| 寶石 | SG 範圍 |
|------|---------|
| 鑽石 (Diamond) | 3.52 |
| 紅寶石/藍寶石 (Corundum) | 4.00 |
| 祖母綠 (Emerald) | 2.72 |
| 翡翠 (Jadeite) | 3.34 |
| 水晶 (Quartz) | 2.65 |
| 立方氧化鋯 (CZ) | 5.80 |

## 🔷 第五部分：偏光儀 (Polariscope)

### 什麼是偏光儀

偏光儀用於確定寶石的光學性質，判斷其為均質體還是非均質體，並觀察干涉圖 (Interference Figure)。

### 光學性質識別

| 觀察結果 | 光學性質 | 寶石類型 |
|----------|---------|----------|
| 全暗（旋轉時無變化） | 均質體 (Isotropic) | 鑽石、玻璃、石榴石、立方氧化鋯 |
| 四明四暗（旋轉時交替變化） | 非均質體 (Anisotropic) | 紅寶石、藍寶石、祖母綠、水晶 |
| 複雜圖案 | 非均質體 + 偏振異常 | 翡翠（經處理） |
| 全亮（無消光） | 無法確定 | 需要其他測試 |

### 干涉圖類型

| 干涉圖類型 | 光軸數 | 寶石例子 |
|-----------|--------|----------|
| 單軸干涉圖 (Uniaxial) | 1個光軸 | 紅寶石、藍寶石、石英 |
| 雙軸干涉圖 (Biaxial) | 2個光軸 | 翡翠、拓帕石、長石 |

## 🔷 第六部分：二色鏡 (Dichroscope)

### 什麼是二色鏡

二色鏡是用於觀察寶石二色性 (Pleochroism) 的儀器。非均質體寶石對不同方向的光有不同的吸收，從而在不同結晶方向顯示不同顏色。

### 常見寶石的二色性

| 寶石 | 二色性描述 | 二色性強度 |
|------|-----------|------------|
| 紅寶石 (Ruby) | 紅色 / 橙紅色 | 強 |
| 藍寶石 (Sapphire) | 藍色 / 藍綠色 | 強 |
| 祖母綠 (Emerald) | 綠色 / 黃綠色 | 中等 |
| 海藍寶石 (Aquamarine) | 藍色 / 淡藍色 | 弱 |
| 紫水晶 (Amethyst) | 紫色 / 淡紫色 | 弱 |
| 翡翠 (Jadeite) | 無明顯二色性 | 無 |

### 實用練習

| 對比 | 結果 |
|------|------|
| 紅寶石 vs 紅尖晶石 | 紅寶石有二色性，尖晶石沒有 |
| 藍寶石 vs 藍色鋯石 | 藍寶石有二色性，鋯石沒有 |
| 祖母綠 vs 翡翠 | 祖母綠有二色性，翡翠沒有 |

## 🔷 第七部分：顯微鏡與放大鏡

### 10x 放大鏡 (10x Loupe)

**標準要求：**
- 放大倍數：10x
- 無色差 (Achromatic)
- 無畸變 (Distortion-free)
- 直徑：20-30mm

### 顯微鏡照明技術

| 照明方法 | 顯示內容 |
|----------|----------|
| 反射光 (Reflected Light) | 表面特徵、刻面磨光 |
| 穿透光 (Transmitted Light) | 整體顏色、內含物 |
| 暗場照明 (Dark Field) | 亮背景上的暗內含物 |
| 亮場照明 (Bright Field) | 暗內含物在亮背景 |
| 偏振光 (Polarized) | 干涉色、應力 |
| 斜照明 (Oblique) | 表面紋理、刻面角度 |

### 常見內含物

| 內含物 | 寶石 | 意義 |
|--------|------|------|
| 針狀內含物 (Needle) | 紅寶石 | 絲狀金紅石針 |
| 八面體晶體 | 鑽石 | 石榴石、橄欖石 |
| 兩相內含物 | 祖母綠 | 液體+氣體 |
| 羽狀裂紋 | 翡翠 | 雷射處理痕跡 |

## 🔷 第八部分：紫外光燈 (UV Light)

### 螢光 vs 磷光

| 類型 | 定義 |
|------|------|
| **螢光 (Fluorescence)** | 當紫外光照射時發光，停止照射後立即消失 |
| **磷光 (Phosphorescence)** | 當紫外光照射時發光，停止照射後仍持續發光一段時間 |

### 常見寶石的螢光

| 寶石 | 長波 UV (365nm) | 短波 UV (254nm) |
|------|-----------------|-----------------|
| 鑽石 | 藍、黃、綠、橙、無 | 藍、黃、綠、橙、無 |
| 紅寶石 | 強紅色 | 強紅色 |
| 祖母綠 | 通常無或有弱綠色 | 視來源而定 |
| 翡翠 | 通常無 | 視處理而定 |

### 實用應用

- **鑽石：** 螢光強度影響價值（強螢光可能降低價值）
- **紅寶石：** 緬甸紅寶石有強紅色螢光
- **翡翠：** B 貨翡翠（充填處理）通常有強螢光

## 🔷 第九部分：寶石鑑定流程

### 系統化鑑定步驟

| 步驟 | 內容 |
|------|------|
| **第一步：肉眼觀察** | 顏色、光澤、透明度、表面特徵 |
| **第二步：放大檢查 (10x loupe)** | 表面特徵、磨損痕跡、內含物概況 |
| **第三步：折射率測量** | 測量 RI 值，判斷 SR 或 DR |
| **第四步：偏光儀檢查** | 確定均質或非均質 |
| **第五步：比重測定** | 水重法測量 SG 值 |
| **第六步：其他儀器測試** | 光譜儀、二色鏡、UV 燈、顯微鏡 |
| **第七步：綜合判斷** | 對比所有觀測結果，得出結論 |

### 常見寶石快速識別表

| 寶石 | RI | SG | 光學 | 螢光 | 特徵 |
|------|----|----|------|------|------|
| 鑽石 | 2.417 | 3.52 | 均質 | 藍(常有) | 無內含物/極少 |
| 紅寶石 | 1.76-1.77 | 4.00 | 非均質 | 紅色 | 絲狀針 |
| 藍寶石 | 1.76-1.77 | 4.00 | 非均質 | 視色 | 色帶/指紋 |
| 祖母綠 | 1.58-1.59 | 2.72 | 非均質 | 通常無 | 三相內含物 |
| 翡翠 | 1.66-1.68 | 3.34 | 非均質 | 通常無 | 纖維交織結構 |
| 水晶 | 1.54-1.55 | 2.65 | 非均質 | 通常無 | 針狀/液體內含物 |
