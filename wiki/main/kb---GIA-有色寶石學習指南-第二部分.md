## 🔷 第一部分：有色寶石概論

### 有色寶石 vs 4C 權重

| 特徵 | 鑽石中的重要性 | 有色寶石中的重要性 |
|------|---------------|-------------------|
| Color (顏色) | ★★★☆☆ | ★★★★★ |
| Clarity (淨度) | ★★★★☆ | ★★★☆☆ |
| Cut (切工) | ★★★★☆ | ★★★☆☆ |
| Carat (克拉) | ★★★☆☆ | ★★★★☆ |

對於大多數有色寶石，顏色佔整體價值的 **50-80%**。

### 顏色三要素

| 要素 | 定義 |
|------|------|
| **色調 (Hue)** | 基本顏色：紅、橙、黃、綠、藍、紫 |
| **飽和度 (Saturation)** | 顏色的純淨程度或強度 |
| **亮度 (Tone)** | 顏色的深淺程度 |

### 顏色成因

| 元素 | 產生的顏色 |
|------|-----------|
| 鉻 (Chromium) | 紅色、粉紅色 |
| 鐵 (Iron) | 黃色、綠色、藍色 |
| 銅 (Copper) | 藍綠色 |
| 錳 (Manganese) | 粉紅色、橙色 |

## 🔷 第二部分：剛玉家族 (Corundum)

### 紅寶石 (Ruby)

**定義：** 紅色剛玉，顏色來自鉻元素。只有紅色到帶輕微紫色/橙色的剛玉才能被稱為「紅寶石」。

#### 主要產地

| 產地 | 特色 | 市場地位 |
|------|------|----------|
| 緬甸 (Myanmar) | 頂級「鴿血紅」，強熒光 | 最優質、最昂貴 |
| 莫桑比克 (Mozambique) | 近年最大產量，品質優良 | 高端市場 |
| 泰國 (Thailand) | 深色但色調均勻 | 中高端市場 |
| 斯里蘭卡 (Sri Lanka) | 淺色調，常有粉紅色 | 中端市場 |
| 越南 (Vietnam) | 品質優異但供應有限 | 高端市場 |

#### 「鴿血紅」(Pigeon Blood Red) 特徵
- 飽和度極高的紅色
- 微微偏藍（經認證的緬甸紅寶石）
- 在所有光照條件下都呈現明亮、均勻的紅色
- 強烈的熒光反應

#### 品質評估

| 因素 | 重要性 | 說明 |
|------|--------|------|
| 顏色 | 50-70% | 主色調為紅色，飽和度高到極高 |
| 淨度 | - | 大多數紅寶石都有內含物，常見金紅石針狀物 |
| 切工 | - | 橢圓形最常見，避免過淺或過深 |
| 克拉 | - | 1ct以上稀有，3ct以上極為罕見 |

#### 「有燒」vs「無燒」
- 熱處理可以改善顏色和淨度
- 經處理的紅寶石比未處理的同等品質 **便宜 30-50%**

---

### 藍寶石 (Sapphire)

**定義：** 除紅寶石（紅色剛玉）以外，所有其他顏色剛玉的總稱。

#### 顏色種類

| 顏色名稱 | 描述 | 市場地位 |
|----------|------|----------|
| 皇家藍 (Royal Blue) | 深藍色，略帶紫色調 | 最有價值 |
| 矢車菊藍 (Cornflower Blue) | 中等藍色，明亮 | 高端 |
| 帕帕拉恰 (Padparadscha) | 橙粉色蓮花色 | 極稀有高端 |
| 粉色藍寶石 | 粉紅色 | 中高端 |
| 黃色藍寶石 | 黃色 | 中端 |
| 白色藍寶石 | 無色 | 低端 |

#### 主要產地

| 產地 | 特色 | 市場地位 |
|------|------|----------|
| 喀什米爾 (Kashmir) | 絲絨感的藍色，極稀有 | 最頂級、最昂貴 |
| 緬甸 (Myanmar) | 深藍色，強熒光 | 高端 |
| 斯里蘭卡 (Sri Lanka) | 明亮藍色，色調多樣 | 高端 |
| 馬達加斯加 (Madagascar) | 近年重要產地 | 中高端 |
| 泰國 (Thailand) | 深色，常需加熱處理 | 中端 |

## 🔷 第三部分：綠柱石家族 (Beryl)

### 祖母綠 (Emerald)

**五大名貴寶石之一**，與紅寶石、藍寶石、鑽石、金綠寶石並列。

#### 主要產地

| 產地 | 特色 | 市場地位 |
|------|------|----------|
| 哥倫比亞 (Colombia) | 最優質，濃郁綠色，「木佐綠」 | 最高端、最昂貴 |
| 贊比亞 (Zambia) | 深綠色，略帶藍色調 | 高端 |
| 巴西 (Brazil) | 較淺的綠色，淨度較好 | 中高端 |
| 巴基斯坦 (Pakistan) | Swat山谷，優質但供應有限 | 高端 |
| 阿富汗 (Afghanistan) | Panjshir山谷，優質但供應不穩定 | 高端 |

#### 「花園」(Jardin)
祖母綠內含物豐富，被浪漫地稱為「花園」(Jardin)。這些內含物是天然祖母綠的證明。

**常見內含物：** 三相/兩相內含物、指紋狀內含物、晶體內含物、裂縫

#### 處理方式
- **傳統油浸處理：** 使用雪松木油填充開放性裂縫
- **聚合物/環氧树脂填充：** 更穩定的現代方法

#### A/B/C 貨
| 類別 | 定義 | 價值 |
|------|------|------|
| A貨 | 天然翡翠，只經過表面拋光和上蠟 | 最高 |
| B貨 | 經酸洗去除雜質，注入聚合物 | 中等 |
| C貨 | 經人工染色 | 低 |
| B+C貨 | 同時經過漂白、填充和染色 | 最低 |

---

### 海藍寶石 (Aquamarine)

**名稱源自：** 拉丁語「aqua marina」= 海水

| 項目 | 內容 |
|------|------|
| 顏色成因 | 鐵 (Iron) |
| 主要產地 | 巴西、尼日利亞、贊比亞、巴基斯坦 |
| 市場定位 | 比祖母綠便宜，是受歡迎的入門級彩色寶石 |
| 克拉大小 | 常見大克拉數，5-20克拉很常見 |
| 熱處理 | 市場上大多數經熱處理去除綠色調 |

---

### 摩根石 (Morganite)

| 項目 | 內容 |
|------|------|
| 顏色成因 | 錳 (Manganese) |
| 名稱來源 | 美國金融家 J.P. Morgan |
| 市場定位 | 中等價格，彩色寶石愛好者的熱門選擇 |
| 特點 | 柔和浪漫的粉紅色，適合訂婚戒指 |

## 🔷 第四部分：其他重要有色寶石

### 翡翠 (Jadeite)

**在中華文化中具有至高無上的地位：8000年玉文化歷史**

#### 類型（按透明度/結構）

| 類型 | 描述 |
|------|------|
| 玻璃種 | 透明度最高，如玻璃般清澈 |
| 冰種 | 半透明，像冰塊 |
| 糯種 | 微透明，質地如糯米飯 |
| 豆種 | 不透明，可見顆粒狀結構 |

#### 按顏色分類

| 顏色 | 說明 |
|------|------|
| 帝王綠 (Imperial Green) | 最珍貴，濃郁、均勻的翠綠色 |
| 蘋果綠 | 明亮的綠色，帶黃色調 |
| 白底青 | 白色底色，帶綠色脈紋 |
| 紫羅蘭 | 紫色翡翠 |
| 墨翠 | 黑色但透光呈綠色 |

#### 主要產地
緬甸（幾乎是唯一的高品質翡翠來源）

---

### 電氣石 (Tourmaline)

**名字源自僧伽羅語 "turamalli" = 「混合顏色的石頭」**

#### 顏色種類

| 顏色/名稱 | 描述 | 市場地位 |
|-----------|------|----------|
| 盧比萊 (Rubellite) | 紅色到粉紅色電氣石 | 高端 |
| 帕拉伊巴 (Paraiba) | 霓虹藍到綠藍色，含銅 | 極高端 |
| 西瓜電氣石 | 粉紅色和綠色同時存在 | 中高端 |
| 靛藍電氣石 | 藍色電氣石 | 中高端 |
| 綠色電氣石 | 綠色電氣石 | 中端 |

#### 帕拉伊巴電氣石
- **顏色：** 霓虹藍、藍綠色，極為鮮豔
- **成因：** 含銅 (Copper) 和錳 (Manganese)
- **價格：** 每克拉可達數千至數萬美元
- **產地：** 巴西帕拉伊巴州（現在已很少開採）

---

### 坦桑石 (Tanzanite)

**1967年於坦桑尼亞北部發現，是相對較新的寶石品種**

| 項目 | 內容 |
|------|------|
| 礦物 | 黝簾石 (Zoisite) 的藍紫色變種 |
| 顏色 | 藍色到紫色，強多色性 |
| 熱處理 | 市場上幾乎所有坦桑石都經過熱處理 |
| 特點 | 只在坦桑尼亞一個地方發現，礦床有限 |
| 保養 | 需要避免超聲波清洗和高溫 |

---

### 歐泊 (Opal)

**非晶質寶石，由矽膠球體規則排列產生變彩 (Play-of-Color)**

#### 類型

| 類型 | 描述 | 市場地位 |
|------|------|----------|
| 黑歐泊 (Black Opal) | 深色背景，變彩鮮豔 | 最高端 |
| 白歐泊 (White Opal) | 淺色背景，變彩柔和 | 中高端 |
| 火歐泊 (Fire Opal) | 橙色到紅色體色 | 中高端 |
| 晶質歐泊 (Crystal Opal) | 半透明，變彩明顯 | 中端 |
| 礫背歐泊 (Boulder Opal) | 與母岩共存 | 中端 |

## 🔷 第五部分：處理方式

| 處理方式 | 說明 | 常見應用 |
|----------|------|----------|
| 熱處理 (Heat Treatment) | 加熱改善顏色和淨度 | 藍寶石、紅寶石、坦桑石 |
| 油浸處理 (Oil Treatment) | 使用油填充裂縫 | 祖母綠 |
| 裂縫填充 (Fracture Filling) | 使用玻璃/聚合物填充 | 紅寶石、祖母綠 |
| 染色處理 (Dyeing) | 使用染料改變顏色 | 翡翠B+C貨 |

## 📝 模擬試題精選

### 選擇題

**Q1: 以下哪個不是祖母綠的主要產地？**
- A) 哥倫比亞
- B) 贊比亞
- C) 緬甸 ❌ (緬甸是紅寶石和藍寶石的主要產地)
- D) 巴西

**Q2: 關於帕拉伊巴電氣石？**
- A) 顏色來自鉻元素
- B) 是電氣石家族中最昂貴的成員 ✅
- C) 名稱來自巴西一個州 ✅
- D) 不含任何微量元素

**Q3: 翡翠市場上的「A貨」是指？**
- A) 質量最好的翡翠
- B) 經過酸洗和聚合物填充的翡翠
- C) 天然翡翠，只經過表面拋光 ✅
- D) 人造翡翠

### 填充題

1. 紅寶石的紅色來自於微量元素 **鉻 (Chromium)**
2. 海藍寶石的顏色來自於微量元素 **鐵 (Iron)**
3. 翡翠市場上，B貨翡翠經過 **酸洗** 處理
4. 「矢車菊藍」和「皇家藍」是形容 **藍寶石** 的最佳顏色描述

### 是非題

1. **對** - 祖母綠的「花園」(Jardin) 是天然祖母綠的特徵之一
2. **錯** - 市場上幾乎所有的坦桑石都經過熱處理
3. **對** - 歐泊的變彩效應是由於光線穿過規則排列的矽膠球體
