### 2️⃣ 五大要點

**① 三機分工係資產，唔係負擔**
作者家裡有三台 Mac，新機唔用 Migration Assistant 全量克隆，保持環境乾淨。新機最大價值在於「零數字債」的乾淨狀態。

**② 先裝代理再裝一切（血淚教訓）**
没在第一时间装代理工具，结果到 brew 这一步被卡了 30 分钟。**终端 CLI 代理要手动 export**，系统代理对 CLI 无效：
```bash
export HTTP_PROXY=http://127.0.0.1:7897
export HTTPS_PROXY=http://127.0.0.1:7897
export ALL_PROXY=socks5://127.0.0.1:7897
```
