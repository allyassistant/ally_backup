## 摘要

你平時用瀏覽器能干的事儿——刷B站、刷推特、刷知乎、刷小紅書——它都能讓你用命令行來幹。

**2. 點解運作？**

```
你輸入命令 → OpenCLI CLI → 守護進程 → Chrome擴展 → 瀏覽器 → 網頁
```
• Chrome 擴展：和瀏覽器「握手」
• 守護進程：後台小服務
• CDP（Chrome DevTools Protocol）：Chrome官方留嘅「後門」
• **關鍵亮點**：復用你已登錄的狀態，唔需要搞API Key、Cookie、驗證碼
