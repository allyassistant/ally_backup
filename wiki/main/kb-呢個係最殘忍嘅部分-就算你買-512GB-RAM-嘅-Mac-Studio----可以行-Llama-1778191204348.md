## 摘要

呢個係最殘忍嘅部分。就算你買 512GB RAM 嘅 Mac Studio：
- 可以行 Llama 405B Q4 量化版（約 200GB RAM）
- Inference speed 大概 5-15 tokens/sec（睇 quantization level）
- 而 Claude Code / Codex 係即時出

仲有——本地行 405B 嘅質量，大概等於 GPT-4 級別。但 OpenAI/Anthropic 最 top 嘅 model（Claude Opus 4、GPT-5 果啲）本地完全跑唔到。你俾咗 10 萬，但攞到嘅係「兩年前」嘅能力。

**避險邏輯有幾個漏洞**

1. **開源都有成本** — 就算你自己跑，電費、散熱、hardware 折舊都係錢。你唔係「唔使俾錢」，你係「將 recurring cost 變做 upfront cost + 電費」

2. **Hardware 係 sinking cost** — 訂閱唔鍾意可以停。Hardware 買咗就用咗，賣出去蝕到喊。

3. **開源 model 唔係免費俾你** — 你可以自己 run，但佢哋嘅 distribution license、API terms、訓練數據來源，全部都有灰色地帶。你用得放心？
